HTML - Linguagem de marcação de hipertexto

Estrutura Básica

Documentos HTML são divididos em algumas partes básicas. Antes de iniciar os elementos, devemos usar uma declaração < > na primeira linha.

O elemento raíz “html” é de onde partem todos os outros elementos, ele é composto das partes “head” e “body”.

Tags e elementos

marcação HTML, cada um é preenchido por uma tag de abertura, conteúdo e uma tag de. Tags de abertura são escritas no formato <nome-da-tag> e tags de fechamento no formato </nome-da-tag>

Elementos de texto

Alguns elementos de texto que podemos usar:

  • parágrafo - tag <p>
  • títulos - tags <h1> a <h6>
  • links - tag <a>

Criando uma nova página

Para criar uma nova página HTML devemos:

  1. Criar um novo arquivo com extensão .html
  2. insira a declaração doctype na primeira linha
  3. Criar os elementos html, head e body
  4. inserir o conteúdo
